2. Chips?!  Can you eat them?!				[CINFO]
==============================================================
No you cannot!  Unless you like to eat plastic/metal. o_O.

Battle chips are the basis of the MMBN series.  All of you should know what
they are.  There are many different type of chips in the game, from Recover
chips, to damage chips.  Each chip has its own purpose that can help you
in a cinch.  A group of 30 chips makes up a folder.  Your folder should be
a group of chips that are based around one common strategy.  Thats
what a good folder is all about.

There are 3 different things that classify a chip:

1. Class
Each chip is part of a large group of chips called a class.  In BN5, there 
are 5 different classes: Standard, Mega, Giga, Secret, and Dark.  In each folder
there are restrictions to the amount of each class chips are allowed in the 
folder.
- Standard - unlimited
- Mega - 5
- Secret - they count as mega or giga chips
- Giga - 1
- Dark - 3  

2. Code
Each chip has a letter code assigned to it.  Codes are very important when 
building a folder.  The more chips with the same code in your folder, the 
faster and more fluid your folder would be.  When making a folder, 
make sure to try and pick chips within only one chip code, unless you want to
be flamed on the boards with the "OMGWTFZ YOU SUXXORZ!!!1!!.

3. Type
Each chip has a certain type to it.  The type can be considered like an
element.  There are many different types of elements in BN5, listed below:
Normal, Fire, Aqua, Wood, Elec, Wind, Recovery, Invisibility, Variable, 
Break, Sword, Obstacle, and Aiming.  Depending on the type of chip
sacrificed at the beggining of a Netbattle turn, a different Soul would
appear fused with Megaman.  Different type chips also have special
effects on other elements.  Such as Elec chips do double damage
on Ocean Panels.
